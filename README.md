# 💳 Credit Card Validator & Generator

Zouti edikatif ki demontre **Algoritm Luhn** (ISO/IEC 7812-1).

## 🎯 Fonksyonalite
| Tab | Fonksyon |
|-----|----------|
| 🔍 Validasyon | Verifye si yon nimewo pase test Luhn |
| ⚙️ Jenere | Jenere nimewo valab selon Luhn |
| 📦 Batch | Verifye plizye nimewo an menm tan |
| 📖 Info BIN | Idantifye bank/peyi selon premye 6 chif |

## 📚 Algoritm Luhn
Yon checksum matematik ki verifye foma yon nimewo kat — pa verifye si kat la egziste.
